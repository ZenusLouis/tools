# Codex Adapter

This folder contains repo-local Codex artifacts generated by GCS Dashboard.

## Layout

- `skills/`: role and workflow skills in Codex `SKILL.md` format.
- `commands/`: command notes that map shared GCS command names to Codex CLI usage.
- `learning/`: Codex-specific lessons and behavior notes.

Shared source-of-truth data lives in `agents/`.
