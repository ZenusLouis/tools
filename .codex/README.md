# Codex Adapter

This folder contains repo-local Codex artifacts generated by GCS Dashboard.

## Layout

- `skills/`: role and workflow skills in Codex `SKILL.md` format.
- `commands/`: command notes that map shared GCS command names to Codex CLI usage.
- `learning/`: Codex-specific lessons and behavior notes.

Shared source-of-truth data lives in `agents/`.

## Dashboard logging

Run Codex through the local GCS wrapper when you want sessions to show on the dashboard:

```powershell
$env:DASHBOARD_URL="https://gcs-dashboard.zenus.dev"
$env:BRIDGE_TOKEN="<token from Settings>"
$env:GCS_PROJECT="dashboard"
powershell -File d:\GlobalClaudeSkills\hooks\codex-gcs.ps1 "Implement task M1-F1-T1"
```

The wrapper sends a bridge heartbeat, runs `codex`, and posts a Codex session log to `/api/log`.
