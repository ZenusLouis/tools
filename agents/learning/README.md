# Agent Learning

This folder is for learning that should be visible across agents.

## Files

- `global-lessons.md`: general lessons that apply to every project and provider.
- `provider-notes.md`: differences between Claude, Codex, and ChatGPT behavior.
- `role-feedback.md`: lessons tied to BA, Dev, Reviewer, QA, Research, or Design roles.

Project-specific decisions still belong in `projects/<project>/decisions.md`.
