# learn

## Purpose

Capture a reusable lesson after a bug, mistake, decision, or repeated project pattern.

## Routing

- General lessons: `agents/learning/global-lessons.md`
- Provider behavior: `agents/learning/provider-notes.md`
- Role behavior: `agents/learning/role-feedback.md`
- Project decisions: `projects/<project>/decisions.md`

## Default Role

`knowledge-curator`
