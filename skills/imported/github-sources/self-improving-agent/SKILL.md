# Self-Improving Agent

Use for learning loops across agents.

## Upstream Sources

- https://github.com/jdrhyne/agent-skills
- https://github.com/alirezarezvani/claude-skills

## Workflow

1. Capture the concrete mistake or repeated pattern.
2. Decide destination: global, provider, role, or project-specific learning.
3. Deduplicate before adding.
4. Write short lessons with date, context, and prevention rule.
5. Promote stable lessons into role rules only after repeated use.
