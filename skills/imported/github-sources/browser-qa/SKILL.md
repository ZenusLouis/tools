# Browser QA

Use for UI smoke tests and visual checks.

## Upstream Sources

- https://github.com/mxyhi/ok-skills
- https://github.com/kodustech/awesome-agent-skills

## Workflow

1. Identify the target route and expected behavior.
2. Use an existing browser automation tool if the repo already has one.
3. Capture screenshots for layout regressions.
4. Report findings in `review.md` with route, viewport, reproduction steps, and expected fix.
