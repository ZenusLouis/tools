# Accessibility Auditor

Use for frontend accessibility reviews.

## Upstream Sources

- https://github.com/kodustech/awesome-agent-skills
- https://github.com/mxyhi/ok-skills

## Workflow

1. Check semantic structure and interactive control labels.
2. Verify keyboard navigation and visible focus.
3. Check contrast and disabled/error states.
4. Prefer accessible native controls over custom widgets unless needed.
5. Report issues with route/component and fix recommendation.
